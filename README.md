# pranay-o.github.io

Personal portfolio site for Pranay Oza — embedded systems engineer.

Built as a static HTML/CSS/JS site with no build step. Hosted on GitHub Pages.

**Live:** https://pranay-o.github.io

## Local preview

Open `index.html` directly in a browser, or run a quick static server:

```sh
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Structure

- `index.html` — markup
- `styles.css` — design tokens + layout
- `script.js` — scroll-triggered fade-ins and mobile nav
- `assets/` — favicon and any future images
